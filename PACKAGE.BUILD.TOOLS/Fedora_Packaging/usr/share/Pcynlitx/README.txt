Pcynlitx has been developed by Erkam Murat Bozkurt as a research study. Basicly,Pcynlitx is an open source meta-program which produces a class library for C++ multi-thread programming and the outcome of the Pcynlitx acts as an autonomous management system for the thread synchronization tasks. The copyright application for the Pcynlitx sofware has been performed in 12 March 2018 to US copyright office and the sofware is ready to use. Pcynlitx has been licenced with GNU General Public License as published by the Free Software Foundation, either version 3 of the License, or any later version. The file includes the thers of use and the license file are included in the software package.

Pcynlitx currently supports only GNU/LINUX operating system and it can be installed on Ubuntu and Debian distros. Pcynlitx will be available for the other operating systems as soon as possible.In order to install PCYNLITX, please follow the instruction given below.

    1-)  Download install script from files section 

    2-)  Open terminal and go to directory in which install script places

    2-)  Then, enter the following commands to the terminal

         -> For Debian users, bash Debian_Install_Script.sh

         -> For Ubuntu users, bash Ubuntu_Install_Script.sh

	 -> For Fedora users, bash Fedora_Install_Script.sh
